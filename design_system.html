CHAPTER 01

🌟ブランド

私たちのアプリは「学生のひらめきを世界へ」がコアメッセージ。それを 閃 という1文字、 クリーム×ヴァーミリオン×サン×インク の4色、 そして紙のステッカー的な質感で表現しています。

プロダクトマーク

ブランドの中心は 「閃」 の一文字。RocknRoll One で組み、 黒地サン色の角丸正方形に -6deg の傾きを付け、 バーミリオン＋サン色の二段オフセット影 で立体感を作ります。 右上には小さなバーミリオンのドット（クリームの縁取り付き）でアクセント。

閃

    8px 8px 0 var(--vermillion),

ボイス＆メッセージ

伝え方の指針。すべてのコピー・通知・ボタンラベルは、この性格でチェック：

— TONE PRINCIPLES
① 学生らしい：難しい敬語や専門用語は避け、友達同士の会話のように軽やかに。
② 短く言い切る：「〜と思います」より「〜です」。曖昧さを残さない。
③ 数字で具体的に：「たくさん」より「24,800人」。リアル感が信頼を作る。
④ 絵文字を使うけど散らかさない：1メッセージに最大1〜2個まで。

CHAPTER 05

🔘ボタン

階層を意識した4種類のボタン。CTAは 1画面に1つだけ がルール。 ホバーで translateY(-2px) 浮き上がる動きが、押せる感を出します。

ボタンスタイル

/* プライマリ：1画面1つ、最重要アクション */
.btn-primary {
  background: var(--vermillion);
  color: #fff;
  border: 2px solid var(--ink);
  border-radius: 14px;
  padding: 12px 18px;
  font-weight: 800; font-size: 13px;
  box-shadow: 4px 4px 0 var(--ink);
  transition: transform .15s ease;
}
.btn-primary:hover { transform: translateY(-2px); }

/* セカンダリ Ink：補助だけど目立たせたい */
.btn-ink {
  background: var(--ink);
  color: var(--sun);
  box-shadow: 4px 4px 0 var(--vermillion);
}

使い分け

— USE WHEN
Primary（バーミリオン）：送信、登録、購入、コラボ承認など、その画面で 最も重要な1つ のアクション。
Ink Secondary（黒地サン）：補助的だが目立たせたいアクション。「次へ」「保存」など。
Paper Sub（紙地黒）：キャンセル、戻る、控えめな選択肢。
Icon Button：ヘッダー・ナビゲーション内のシンボル単体。

CHAPTER 08

🧩パターン

コンポーネントを組み合わせた、繰り返し使う「型」。新しい画面を作るときは、まずこの型から組み立てます。

セクション見出しパターン

全画面で共通のセクション開始の合図。キッカー（小さい英字ラベル）→ 見出し → 説明 の3段構造。 キッカーは — [LABEL] 形式の英字＋ダッシュで、サン色の蛍光ペン強調を見出しに1つ。

LATEST

新着のアイディア

毎日、全国の大学から100件以上のアイディアが投稿されています。

ボトムCTA（モバイル）

モバイル画面の底部に固定するアクションバー。背景は紙、上に2px黒線。

— PATTERN RULE
・常に2px の上線で本文と分離
・余白は 12px 18px 24px（下を厚めに：iPhoneのホームインジケータを意識）
・3つ以上のボタンを並べないこと

空状態 / エラー状態

— EMPTY STATE
中央寄せ、絵文字（48-56px）、見出し（900）、説明（1行）の4要素。
例：📭 該当する通知はありません ／ 別のカテゴリを選んでみてください

確認モーダル

— MODAL DESIGN
・絵文字（48-60px）が常に最初
・見出し（900）→ 説明（1.65 line-height）→ プライマリ → セカンダリの順
・bounceIn で 0.55s かけて弾むように出現
・破壊的アクション（削除など）は影をバーミリオンに
CHAPTER 09

🎬モーション

動きにも法則があります。速くて軽い が基本（150〜350ms）。重い動きは特別な瞬間にだけ使います。

タイミング

--motion-fast120ms / リアクション
--motion-base150ms / 基本
--motion-medium250ms / 状態切替
--motion-slow350-550ms / モーダル登場
--motion-feature900ms+ / 特別演出

イージング

基本は ease または cubic-bezier(.5, 1.5, .5, 1) の弾むカーブ。

/* 基本 — UI状態切替 */
transition: all .15s ease;

/* 弾む — モーダル登場、成功演出 */
animation: bounce .55s cubic-bezier(.5, 1.5, .5, 1);

/* 浮く — ホバー時の沈み */
.btn:hover { transform: translateY(-2px); }

/* 揺れる — 永続アニメーション */
@keyframes float {
  0%, 100% { transform: translateY(0) rotate(-6deg); }
  50% { transform: translateY(-6px) rotate(-6deg); }
}

よく使うアニメーション

01

Hover Float

ボタン・カードを150msで translateY(-2px)。「押せる感」の基本。

02

Bounce In

モーダル・成功画面で550msかけて scale(.5)→1。喜びの演出。

03

Pulse

未読ドット・ライブ表示などで1.4sの呼吸。「生きてる感」。

04

Shake

無効入力時に350ms左右ブレ。優しい否定のフィードバック。

設計原則（4つの哲学）

01

触ってわかる「ステッカー感」

2px縁取り＋3-4pxオフセット影でリソグラフ印刷の質感を再現。すべてのカード型要素に適用。

02

4色だけで世界を表現

クリーム / インク / バーミリオン / サンを基本に色を絞る。補助色は意味があるときだけ。

03

3つのフォントで役割を分ける

Zen Kaku Gothic / RocknRoll One / Bricolage Grotesque で声色を分ける。役割を交差させない。

04

使った瞬間に笑顔になる細部

紙吹雪、回転テーブル、絵文字の揺れ、ボタンの沈み込み。機能じゃなく感情を設計する。
